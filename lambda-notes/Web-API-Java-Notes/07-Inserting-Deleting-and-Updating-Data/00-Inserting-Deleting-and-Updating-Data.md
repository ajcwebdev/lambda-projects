# Inserting Deleting and Updating Data using CRUD Operations including generating Seed Data


At the end of this module, you should be able to:

1. Perform CRUD operations on an RDBMS JPA and Hibernate (Creating and Updating Records)
2. Perform CRUD operations on an RDBMS JPA and Hibernate (Updating certain fields in records)
3. Perform CRUD operations on an RDBMS JPA and Hibernate (Deleting records)
4. Implement @Transactional annotation
5. Implement a data seeding class using JPA and Hibernate
6. Implement massive data seeding
7. Use H2 Console and H2 IntelliJ integration to explore data
8. Use Postman to manually test Web APIs with all CRUD operations
